JPEG

This is used for compressing images, and it is a "lossless" format. This means the picture doesn't lose quality.

Best use: Digital Photography

PNG

This is used for compressing images, and it is a "lossy" format. This means the picture loses quality when compressed.

Best use: Line Drawing/Text

GIF

This is used for compressing images, and it is a "loseless" format. This image format supports static, and animated images.

Best use: Animations
